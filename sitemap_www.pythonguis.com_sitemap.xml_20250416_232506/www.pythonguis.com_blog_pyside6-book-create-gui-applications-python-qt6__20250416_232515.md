# PySide6 Book now available: Create GUI Applications with Python & Qt6
The hands-on guide to making apps with Python
by [Martin Fitzpatrick](https://www.pythonguis.com/authors/martin-fitzpatrick/) Last updated Feb 28 PySide6 [ blog ](https://www.pythonguis.com/blog/)
Hello! This morning I released the first **Qt6 edition of my PySide book[Create GUI Applications, with Python & Qt6](https://www.pythonguis.com/pyside6-book/)**.
This update follows the 4th Edition of the [PySide2 book](https://www.pythonguis.com/pyside2-book/) updating all the code examples and adding additional PySide6-specific detail. The book contains **600+ pages** and **200+ complete code examples** taking you from the basics of creating PySide applications to fully functional apps.

**PySide6 Book now available: Create GUI Applications with Python & Qt6** was written by [Martin Fitzpatrick](https://www.pythonguis.com/authors/martin-fitzpatrick/) . 
Martin Fitzpatrick has been developing Python/Qt apps for 8 years. Building desktop applications to make data-analysis tools more user-friendly, Python was the obvious choice. Starting with Tk, later moving to wxWidgets and finally adopting PyQt. Martin founded PythonGUIs to provide easy to follow GUI programming tutorials to the Python community. He has written a number of popular [https://www.martinfitzpatrick.com/browse/books/](https://www.pythonguis.com/blog/pyside6-book-create-gui-applications-python-qt6/Python books) on the subject.
