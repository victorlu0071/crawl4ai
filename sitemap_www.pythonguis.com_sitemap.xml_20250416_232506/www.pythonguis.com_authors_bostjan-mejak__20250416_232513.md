#  Boštjan Mejak Python & PySide developer
Boštjan Mejak has been developing Python apps for about ten years. Nothing serious at first, now a proud author of [SuperChess](https://github.com/PedanticHacker/SuperChess), a GUI app for playing chess, written in Python with PySide. He started with Tkinter, then switched to wxPython, PyQt, and recently to PySide.
